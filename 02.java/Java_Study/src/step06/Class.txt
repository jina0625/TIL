클래스 구성요소 (클래스의 멤버)

1. 필드 (= 멤버 변수)
	: 클래스에서 선언하는 변수
	   객체 데이터에 해당하는 상태나 속성 저장 시 사용
	   접근 제한자 사용 가능
	  
	  (1) 인스턴스 변수
	  	: 클래스가 객체 생성할 때마다 새로 생성되는 변수
	   	    객체마다 각각 고유한 값 저장
	   	    클래스에서 선언하지만 객체에서 각각 고유한 값으로 초기화
	   	    
	  (2) 클래스 변수
	  	: 클래스로 생성한 모든 객체에서 공유하는 변수
	  	    선언 시 앞에 static 키워드 붙임 	
	  	    클래스에서 한 번만 초기화하고 모든 객체에서 같은 값 사용
	  	    
	  (3) 지역 변수
	  	: 메서드에 선언하는 변수
	  	    메서드 실행될 동안만 지역 변수에 저장된 값 유효
	  	        
2. 메서드 (= 함수)
	: 클래스 안에 정의한 함수
	  객체가 수행하는 동작이나 기능
	  객체 상태 변경하거나 특정 작업 수행
	  접근 제한자 사용 가능
	  
	  - 반환형
	  	: 메서드 실행한 뒤 결과로 어떤 자료형 값 돌려줄지 선언하는 것
	  	    생략할 수 X (결과 돌려줄 필요 없으면 void 키워드 선언)
	  	    반환값은 메서드 이름 앞에 선언한 반환형과 자료형 같음
	  	    
	  - 매개변수
	  	: 메서드가 외부에서 전달받는 값 저장하는 변수
	  	    자료형은 전달받은 값의 자료형과 같아야 함
	  	    매개변수 없으면 빈 괄호만 표시
	  	    메서드 내에서만 유효한 지역변수
	  	    
	  - 인자
	  	: 매개변수가 있는 메서드를 호출할 때 매개변수에 전달하는 값
	   
3. 생성자
	: 객체 생성하고 초기화
	    객체 생성될 때 자동으로 실행, 생성된 객체 초깃값 설정하는 데 사용
	    클래스와 이름 같고, 반환형 명시 x
	    
	    - 기본 생성자
	    	: 매개변수 없는 생성자
