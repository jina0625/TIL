[자바 변수명 규칙]
1. 숫자 시작 x
2. 특수문자 허용 (_, $)
3. 길이 제한 x
4. 대소문자 구분
5. 예약어 사용 x

[변수명 표기법]
1. 카멜 표기법 (camel case)
	첫 글자는 소문자, 첫 단어 제외 각 단어 첫 글자 대문자로 표기
	int fisrtNumber;
2. 파스칼 표기법 (pascal case)
	모든 단어 첫 글자 대문자, 클래스명 지을 시 사용
	int FirstNumber;
3. 스네이크 표기법 (snake case)
	모든 단어 소문자 표기, 언더바(_)로 단어 연결
	int first_number;
4. 헝가리안 표기법 (Hungarian notation)
	접두어 붙이는 표기법
	int intNumber;
	String strName;
	String txtMyName;
