추상 클래스 (abstract class)
	- 앞에 abstract 키워드 붙여 선언
	- 반드시 추상 메서드 오버라이딩해서 동작 구현해야 함
		(추상 메서드 : 메서드 시그니처만 선언하고, 어떤 동작 하는지 구체적 내용 정의 X)
	- new 키워드 사용해 직접 객체 생성 X
		(객체는 상속받는 자식 클래스에서만 생성할 수 있음)
	- 추상 메서드는 물론 동작까지 정의된 일반 메서드 가질 수 있음
	
[접근 제한자] abstract class 클래스명 {
	// 필드
	// 일반 메서드
	abstract 반환형 메서드명 (매개변수1, 매개변수2...); // 추상 메서드
}
